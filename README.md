# Affiliate Static Starter

Minimal static blog starter using **Astro** + **Tailwind** with Feature-Sliced Design (FSD) structure.

Features:
- Static Site Generation (SSG) with Astro
- Content in `content/blog/*.md`
- Tag filtering, pagination, article pages
- Tailwind CSS + Typography plugin
- Simple article card, tag list, pagination, related articles carousel

How to run:
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   npm run preview
   ```

Notes:
- This repo is a starter. After cloning, create Markdown files in `content/blog/` (examples included).
- Images live in `public/images/...`.
